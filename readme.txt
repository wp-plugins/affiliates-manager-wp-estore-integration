Process an affiliate commission via Affiliates Manager after a WP eStore checkout

== Description ==

This addon allows you to integrate WP eStore with the Affiliates Manager plugin.

When a user pay for an item on your site, this addon will check to see if the user was referred to your site by an affiliate. It will then give commission to the affiliate who referred the user.

This addon requires the [Affiliates Manager](https://wordpress.org/plugins/affiliates-manager/) and [WP eStore](https://www.tipsandtricks-hq.com/wordpress-estore-plugin-complete-solution-to-sell-digital-products-from-your-wordpress-blog-securely-1059) plugins.

Read the usage instructions [here](https://wpaffiliatemanager.com/affiliates-manager-sell-digital-downloads-integration/)

== Installation ==

1. Upload `affiliatemgr-estore-integration` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
